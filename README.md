# NITAC-PCC-website
 
弊高専パソコン部のHPを作りました

url : [https://nitac-pcc-website.web.app/](https://nitac-pcc-website.web.app/)

# Setup
 
```bash
$ git clone https://github.com/Genshi0916/NITAC-PCC-website.git
$ cd NITAC-PCC-website
$ npm install
``` 
# Start

```bash
$ npm run dev
```
